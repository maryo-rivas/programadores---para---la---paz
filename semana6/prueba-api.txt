Al enviar una petición POST a la ruta http://localhost:3000/registro con los datos en formato JSON, el servidor recibe la información enviada por el usuario y responde mostrando los mismos datos recibidos.

Respuesta obtenida:

{
  "estado": "Datos recibidos",
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

Esto demuestra que Express puede recibir datos mediante req.body y devolverlos en formato JSON.
